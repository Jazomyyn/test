######## PRUEBA TÉCNICA - JENIFFER JAZMYN FLORES DESALES #######

Configuración inicial:
    - Crear una base de datos mysql llamada como el DB_DATABASE en el archivo .env (prueba_jjfd)
    - Correr migraciones para generar el esquema (php artisan migrate)
    - Ahora es momento de insertar datos a las base de datos, así que hay que correr los seeders (php artisan db:seed --class="DatabaseSeeder") 
    - (Local) Arrancar la página (php artisan serve)
